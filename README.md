# MIPBC1 - Módulo de Inspeção Preditiva de Baixo Custo

Professor orientador: Dartisson Castro Reis
## Sobre o Projeto
O projeto de Módulo de Inspeção Preditiva de Baixo Custo (MIPBC1) é um protótipo que utiliza sensores de baixo custo para obtenção de dados em tempo real de equipamentos industriais. Por meio de prototipagem rápida utilizando o microchip didático Arduino, criou-se um produto que entrega valor para profissionais que atuam na manutenção mecânica e elétrica diária, inspetores de equipamentos industriais, e colaboradores que trabalham com Planejamento e Controle da Manutenção.
## Especificações Técnicas
| ID | Componente |
| ------------- | ------------- |
| Microcontrolador | Arduino UNO R4 WiFi |
| Sensor de Temperatura | MLX90614 |
| Display | OLED 0,96" SSD1306 |
| Protocolo de Comunicação | I2C + WiFi |
| Plataforma de Dados | ThingSpeak |
| Intervalo de Leitura | 15 segundos |
| Limite Crítico | 60.0 °C
| Tipo de Medição | Temperatura por Infravermelho
